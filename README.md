A 3d Physics Engine written in Rust
# Build Instructions
``cargo run --package ferrum_app --bin ferrum``
# Controls
- WASD -> Move Forward, Left, Down, Right
- Space -> Go Up
- Shift -> Go Down
- Arrow Keys -> Move Camera
- Left Click Drag -> Move Camera
